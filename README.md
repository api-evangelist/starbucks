# Starbucks (starbucks)
Starbucks is a Fortune 500 company that provides developer APIs and integration capabilities through its developer platform. Starbucks provides partner APIs for ordering, loyalty program integration, store locations, and menu data. These APIs enable partners to integrate Starbucks ordering and rewards into their applications.
